Week Overview

This condensed first session covers course logistics and dives directly into 3D modeling fundamentals in Blender. The session concludes with an introduction to the first abstract art assignment and a demonstration of the Array modifier.

Key Concepts

  • Points, edges, and faces are the atomic building blocks of all 3D geometry
  • Object Mode manipulates whole objects; Edit Mode manipulates individual geometry
  • Orthographic views (Numpad 1, 3, 7) provide precise, distortion-free alignment
  • Software-agnostic fundamentals transfer across all 3D applications
  • Array modifier creates procedural repetition from a single object
  • Multiple attendance options: in-person, live stream, or asynchronous

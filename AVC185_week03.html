Key Concepts

  • Quads are preferred topology; N-gons cause shading and subdivision issues
  • Edge loops create clean topology that follows object shape
  • Face normals must point outward (blue=good, red=bad)
  • Extrude, Inset, and Loop Cut are the core mesh editing tools
  • Apply Scale (Ctrl+A) before using modifiers or editing tools
  • Tools edit geometry directly; modifiers are procedural and non-destructive

Key Concepts

  • Global illumination simulates realistic light bouncing in enclosed spaces
  • Weight painting controls scatter distribution density on surfaces
  • Backface culling hides reversed normals (important for game engines)
  • Z-fighting occurs when faces overlap at the same position
  • Sculpting tools create organic shapes by pushing/pulling mesh geometry
  • Subdivision Surface + sculpting creates natural-looking rocks and terrain

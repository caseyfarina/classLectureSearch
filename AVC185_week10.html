Key Concepts

  • Grid fill only works with even numbers of subdivisions around circular objects
  • UV packing allows multiple objects to share one texture set while remaining separate meshes
  • ID maps enable easy masking of individual meshes within shared texture sets
  • Anchor points allow reading layer information across multiple channels in Substance Painter
  • Smart masks combined with manual painting provides procedural generation with artistic control
  • Breaking up computer-perfect surfaces with variation creates more realistic materials
  • Controlling edges in subdivision surface modeling prevents unwanted smoothing
  • Separate layers for different material properties allows independent adjustment of effects

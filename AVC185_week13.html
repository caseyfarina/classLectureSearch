Key Concepts

  • Soft body simulation treats vertices as connectors and edges as springs with varying tension.
  • Topology density is the primary factor in determining how an object deforms during simulation.
  • Physics simulations must be rewound to the start frame to reset properly; scrubbing is unreliable.
  • Self-collisions are disabled by default for performance and must be enabled to prevent mesh clipping.
  • A 'Convert to Mesh' operation freezes a simulation frame into a static asset for easier art direction.
  • Substitute complex model topology with simple primitive colliders to increase simulation stability.
  • Texture fabric patterns on flat, unfolded versions of assets before applying simulated deformations.

Key Concepts

  • High poly meshes are essential for sculpting detail, but must be baked onto low poly meshes for animation and rendering efficiency.
  • The cloth simulation stack order is critical; the Cloth modifier should usually precede the Subdivision Surface modifier for stable results.
  • Volumetric flames can be created using the Principled Volume shader by controlling density and emission with gradient textures.
  • The Blender Compositor allows for GPU-accelerated post-processing, including bloom, glare, and chromatic aberration.
  • Depth of Field effects should be dialed in using the f-stop parameter and the focus eyedropper tool on a specific target object.
  • Using Alt+D to create instances saves significant memory compared to Shift+D duplicates because instances share underlying mesh data.

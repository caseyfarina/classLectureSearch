Key Concepts

  • Cycles uses path tracing to simulate real-world photons for physically accurate lighting.
  • GPU rendering is significantly faster than CPU rendering for graphics-intensive calculations.
  • Denoising allows for high-quality images with dramatically lower sample counts.
  • The Viewport should be set to Solid mode when performing a final render to avoid resource splitting.
  • The Index of Refraction (IOR) dictates how light bends through transparent materials like glass.
  • The Screw modifier is the primary tool for creating radially symmetric objects like wine glasses.
  • Procedural noise nodes and Color Ramps can add 'wobble' and surface grime for realism.
  • A lower F-stop number creates a shallower depth of field and more pronounced bokeh.

Key Concepts

  • Use the Sculpting brush to art-direct cloth meshes after the simulation is frozen.
  • Alt-D creates instances that save VRAM by reading the same memory slot in the GPU.
  • Lattice deformers allow for complex, non-destructive shaping of high-poly meshes.
  • Real-world camera physics: focal length and f-stop both influence the depth of field.
  • LED Volumes ('The Volume') allow for real-time in-camera reflections and lighting for actors.
  • Set camera keyframes to 'Linear' instead of 'Bezier' to avoid unsinematic easing.
  • Render to PNG image sequences to prevent file corruption and allow for post-processing.
  • Tri-planar mapping solves texturing issues on meshes without clean UV unwrap data.

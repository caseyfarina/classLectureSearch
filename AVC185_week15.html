Key Concepts

  • Optimizing 3D models for the web requires converting textures from PNG to JPEG and using GLB/GLTF formats.
  • SketchFab provides an interactive 3D viewer that can be embedded into digital portfolios for interactive student feedback.
  • Non-photorealistic rendering (NPR) uses compositing nodes like Kuwahara and Posterize to create painterly or graphic looks.
  • Grease Pencil Line Art modifiers allow for procedural outlines that update dynamically with the camera view.
  • Cryptomatte is an efficient tool for isolating specific objects in a render for targeted post-processing.
  • Exporting render layers via the File Output node enables high-fidelity editing in external imaging software.

Key Concepts

  • Believable 3D animation requires camera movements that reference real-world cinematography techniques.
  • Cycles rendering provides realistic physics, but material thickness and IOR settings are critical for accurate glass.
  • Denoising tools are effective for stills but can cause 'wobbliness' in animation without sufficient sample counts.
  • Node-based workflows using Color Ramps allow for precise control over surface roughness and specular variation.
  • Resource-saving techniques like 2.5D image planes can be indistinguishable from 3D models at a distance.
  • Optimizing render time can be achieved by setting hard time limits per frame in the Blender render settings.

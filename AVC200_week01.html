Key Concepts

  • Unity uses a component-based architecture: GameObjects + Components
  • Animation Controller manages state transitions between animation clips
  • Smooth loops require matching first and last keyframes
  • Test incrementally ("avoid the big bang") rather than building everything first
  • Scripts add interactivity by responding to player input and events
  • Real-time engines render frames continuously, unlike offline rendering

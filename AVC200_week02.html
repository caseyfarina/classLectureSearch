Key Concepts

  • World space is absolute; local space is relative to parent object
  • Prefabs are reusable templates; changes propagate to all instances
  • HDR emission values above 1.0 create bloom/glow effects
  • Spawners instantiate prefabs at runtime via scripts
  • Modular design: build small reusable pieces, combine into complex systems
  • Time.Destroy removes objects after a set duration (cleanup)

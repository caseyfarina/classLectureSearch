Key Concepts

  • WAV is lossless (larger files); MP3 is compressed (smaller, slight quality loss)
  • Audio Source plays sound; Audio Listener (on camera) receives it
  • 3D sound uses distance falloff; 2D sound plays at constant volume
  • Sorting layers control draw order for 2D sprites in 3D space
  • Blender Delivery Kit ensures clean FBX export with proper scale and origin
  • Normal maps add surface detail without extra geometry

Key Concepts

  • Unity Recorder captures gameplay as video without external screen recording
  • Randomized animation speed creates natural variation in repeated prefabs
  • C# variables store data; functions perform actions on that data
  • Integers are whole numbers; floats handle decimals (important for Unity math)
  • Time.deltaTime makes movement frame-rate independent
  • Unity Events connect mouse clicks and input to game object behaviors

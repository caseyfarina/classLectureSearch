Key Concepts

  • Shader Graph must be saved (Ctrl+S in the graph window) to compile and update the material in the viewport
  • Unlit shader type for hologram: no shadows needed, enables emission for glow
  • Surface type must be set to Transparent to expose the alpha input in Shader Graph
  • Fresnel node: outputs white on edges facing away from camera, black facing toward; controls see-through hologram look
  • Invert the Fresnel output to get transparent center with glowing edges
  • UV packing: Average Island Scale then Pack Islands to fit all objects into one 0-1 UV space
  • Multi-object Substance Painter: keep objects separate (not joined) to allow per-object material assignment in Unity
  • UVs must be established before Mixamo rigging; re-rigging after UV changes breaks material mapping
  • Physics force with tag filtering: create a tag for the target object and match exactly in script (case-sensitive, no spaces)
  • Timer component: set Count Up to false for countdown; use SetActive to disable game objects on timer end

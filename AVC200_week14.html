Key Concepts

  • Use alpha clipping and transparency in Shader Graph to create efficient, realistic foliage and thin objects.
  • Zipping Unity projects correctly requires including all folders EXCEPT the Library folder to keep file sizes manageable.
  • Disable 'Has Exit Time' in the Unity Animator for more responsive character transitions between idle and walking states.
  • Implement third-person camera systems using Cinemachine to avoid clipping and improve environmental visibility.
  • Utilize tag-based trigger zones to communicate between player colliders and global health managers.
  • Enhance environmental atmosphere with volumetric lighting, particle effects, and adaptive music cues.
  • Plan final portfolios to showcase both technical functionality and artistic consistency.

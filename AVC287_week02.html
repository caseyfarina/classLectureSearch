Key Concepts

  • Interpolation fills frames between keyframes; tweening is the traditional animation term
  • Glow effects use compositing nodes to add bloom to emissive materials
  • Ease in/out curves control acceleration and deceleration of motion
  • Frame rate is an artistic choice: 24fps (film), 30fps (video), 12fps (stylized)
  • HD (1920×1080) vs 4K (3840×2160) affects render time and file size
  • Postmodern animation intentionally breaks traditional rules for artistic effect

Key Concepts

  • Blender's grease pencil enables 2D hand-drawn animation in a 3D environment
  • Mass and weight affect how objects move: heavy = slow acceleration, light = quick bounces
  • Bouncing ball is the fundamental animation exercise for learning timing and spacing
  • Rotation animation requires understanding weight distribution and center of mass
  • Drawing tablets provide pressure sensitivity for natural-feeling 2D strokes
  • Combining rotation with translation creates believable hammer physics
